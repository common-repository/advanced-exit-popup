== Description ==
Advanced Exit Popup is a free WordPress plugin developed by <a target="_blank" href="https://wphost.pk">https://wphost.pk</a> that allows you to display custom code like HTML5 / CSS3, Facebook like box, Shortcodes, etc when user intent to exit your website on desktop or mobile.

Advanced Exit Popup automatically detects the mouse behaviors on desktop and scroll behaviour on mobile to show a pop up window which indicates your visitors are about to leave your site.

<strong>Plugin Features</strong>

- Click on close icon image or out of popup window to hide/remove popup
- Unlimited colors for title and body content popup
- WordPress shortcode can be used in popup content
- Control the Height and Width of your Popup window

Up coming version will have many more options.

== Installation ==
1. Upload `advanced-exit-popup.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "Adv Exit Popup" section page.
4. Fill in all fields.
That's all!
